# P2MRT — Photo to Mean Radiant Temperature

Short description
- Convert images / photo data into mean radiant temperature (MRT) estimates.
- Purpose: placeholder project structure for development.

Quickstart
1. Create a virtual environment:
   - python -m venv .venv
   - source .venv/bin/activate (Unix) or .venv\Scripts\activate (Windows)
2. Install:
   - pip install -e .
3. Run tests:
   - pytest

Project layout
- src/p2mrt/: main package
- tests/: unit tests
- .github/: issue & PR templates, CI workflows
